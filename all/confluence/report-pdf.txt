body: # Divert Submit

Request Type | Endpoint  
---|---  
`GET` | _`/divert_report_pdf/cid/<>/jid/<>/drop_date_from/<>/drop_date_to/<>`_  
  
## Steps

Steps are same as the api [Divert Report API](divert/report#steps)  
But instead of sending a JSON response it will respond with a **PDF Document**
if data found in this date range.
