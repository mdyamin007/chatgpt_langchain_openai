### Generic DataCore WebServices

We would like to establish a set of generic web services that make outbound
call to data-cores to fetch necessary information. We used to go through Mule
for the SSCU Pathway setup. But it seems Mule is rather heavy for such simple
use. We will try establish following simple services using Jersey to see if we
gain performance & efficiency over this.
