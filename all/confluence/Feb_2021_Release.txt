title: Feb 2021 Release
body: **Features and Enhancement Release:**

  *  **Email and SMS ([13239](https://app.na3.teamsupport.com/?TicketNumber=13239))- **Allows eStatement users to enroll to receive both email and SMS notification.  Currently our setup only allows user to choose either email or SMS but this update allow users to choose both.
  *  **Non Encrypted PDF Attachment  - **Allows user to enroll to receive an unencrypted PDF attachment via email.  User will not be prompted when the choose PDF attachment notification preference during enrollment process .
  *  **Check Images Through API ([14184](https://app.na3.teamsupport.com/?TicketNumber=14184))- **Update InfoService API to support view check feature that we currently support on our eStatement site.
  *  **User Name Display Alphabetical ([14165](https://app.na3.teamsupport.com/?TicketNumber=14165)) - **Display user name alphabetically under the Account Management module in InfoTRAC. 
  *  **Account Number Display Numerical ([14166](https://app.na3.teamsupport.com/?TicketNumber=14166))- **Display account number numerically under the Account Management module in InfoTRAC. 
  *  **PDF Icon Replacement ([12473](https://app.na3.teamsupport.com/?TicketNumber=12473)) - **Replace PDF Icon on Hypercube's Statement index page and Account Management module in InfoTRAC.
