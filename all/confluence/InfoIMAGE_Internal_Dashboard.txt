## Technology

Front end UI - [Semantic UI React](https://react.semantic-ui.com/)

Javascript UI Library - [ReactJS](https://reactjs.org/) (JSX)

Javascript run-time environment - [NodeJS](https://nodejs.org/en/)

NodeJS Library to connect to OracleDB - [Node
OracleDB](https://github.com/oracle/node-oracledb)

## Objective

Page to plan internal dashboard to provide tools for internal staff so they
can better serve our customers.

## Success metrics

Less bottleneck for ISD to quick tasks.

Goal| Metric  
---|---  
Relief simple tasks off ISD| Small tasks given to ISD reduced  
  
|  
  
  
## Assumptions

  * All users using the internal tools are in the InfoIMAGE network.
  * All users are on desktop, but we will create responsive UI.
