title: Code Release Workflow
body: # Environmentsenvironments

Code is released sequentially into four different environments listed below:

#### 1. Development (Dev)

Code still in the process of being written/fine-tuned.

#### 2\. Pilot (Pilot)

Code is released for client testing.

#### 3\. Staging (Stage)

Codes to undergo QA before being released into Prod.

#### 4\. Production (Prod)

Live codes that are active on client/member site, using real data.
