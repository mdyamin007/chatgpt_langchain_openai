  * All tasks required of a "Resource Upload"
  * Update PUI to be able to process a new prefix under an existing statement type
  * Add client file to file transfer status on PUI

  * All tasks required of a "Resource Upload"

  * Enable selective paperless if required
  * Setup paperless file if required
  * Add client file to file transfer status on PUI

| 3

  * Remove from PUI code and PUI/FTP configurations
  * Remove data (input, pv_nas, peb3)
  * Remove from API, SMS, etc (if necessary)

| 3

  * Remove data (input, pv_nas, peb3)
