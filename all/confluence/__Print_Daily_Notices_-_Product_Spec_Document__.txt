# **Client/InfoIMAGE Fit**  
  
---  
  
#### What is Print Notices?

|

#### Product Overview

This product is the core of our print business. Our clients will have various
print applications in which they will send the data via FTP transmission.
InfoIMAGE will construct this data into its various print applications daily
using the clients' preferred paper and envelope type. We then will presort and
mail out these print notices/letters to clients' customers with proper
tracking and handling.

#### Applications

Notices

Letters

#### Features Include

Setup

  * Input Files for each application

Production

  * Laser Print Notices/Letters

Receive Files

  * Via FTP data transmission

Mailing Service

  * Fold and insert documents

Processing

  * Sort and file IMB barcode
  * Append OMR codes to handle multiple pages
  * Append 2D Glyph for 100% data-to-mail integrity
  * Special extraction for customer return files
  * Special handling of foreign mail 
  * Process data into print format
  * Seal and indicate envelopes
  * Sort to FirstClassCASS certified presort rate on all qualified envelopes
  * Delivery to USPS

  
  
#### Where does Print Notice fit?

|

#### Client's Fit

This provides our clients with a solution for their customers who prefer/ rely
on print daily notices/letters due to location, technology, background, etc.

#### InfoIMAGE's Fit

Daily Notices provides InfoIMAGE a way to provide service on a more frequent
basis compared to monthly or annual statements.
