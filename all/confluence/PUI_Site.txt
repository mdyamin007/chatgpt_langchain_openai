title: PUI Site
body:   * Image below reflects all the PUI Jobs related to Payebill client.  

    1. Auth file Processing - Balance sheet processing
    2. Payebill Processing - Index(Bill) file processing
    3. PreScheduled Payments Processing - Deals with sending payments which are scheduled for that day by autopay or by by request scheduled time.
    4. Autopay Processing - schedules autopayments for new bills
    5. Opt in Opt Out report - The paperless report
    6. Processing Error Report - Summary of Auth and Payebill processing which is sent to the client

  * More technical details about PUI processing can be found in [PayeBill Features](https://i3dev.easyredmine.com/projects/payebill/wiki/PayeBill_Features)
