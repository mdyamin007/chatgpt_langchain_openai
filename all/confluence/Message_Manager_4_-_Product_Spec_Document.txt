#### What is Product?

|

#### Product Overview

InfoIMAGE offers a robust industry leading capability to add targeted on-
statement messages driven by pre-determined selection criteria. Based on 5
standard selective messaging criteria, you will be able to target specific
account holders with messages and embedded graphics (black & white for print,
color for electronic). Statement Messages are easily managed through
InfoIMAGE's InfoTRAC web dashboard.

#### Features Include

  * Easy to Create Templates from Active Criteria
  * Re-useable Targeting Template across Message Manager Campaigns
  * Powerful Content Editing on the Fly
  * Text and HTML Based Editing
  * Re-usable Image Library
  * Floating text messages
  * Across eStatement and Print
