## In Summary

  1. Avoid using document.ready because it might give the user the impression of slow loading speed
  2. Be careful on picking the right iterator
  3. Avoid using "this" as it gets confusing when reading the code and could cause problem during upgrades
  4. Pick only the necessary jQuery libraries just like the important java libraries when using it
  5. Avoid using jQuery if not needed
