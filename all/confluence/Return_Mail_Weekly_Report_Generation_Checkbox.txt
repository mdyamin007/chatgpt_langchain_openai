title: Return Mail Weekly Report Generation Checkbox
body: Configurable weekly report generation for return mail based on client
preference from the User Management module.

Previously return mail weekly report generation was not configurable from the
client management module.

If this feature is selected(checkbox selected), a report will be generated for
that particular client each week. Similarly, if this feature is NOT selected
(checkbox de-selected), no report will be generated for that particular client
each week. This feature will be provided by default, that means, by default
the checkbox will be selected for any client.
