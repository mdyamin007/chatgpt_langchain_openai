title: The Definition of Done
body:   * Automated tests written to the testing pyramid: units tests for business logic, integration tests for entire stack, and UI tests for critical user paths.
  * Code should not have temporary variables/logic. Webpage should not have temporary image/text.
  * No part of the code/webpage should be non-functional, or without purpose. 
  * Code is peer-reviewed by someone else in the team. A different person verified the item worked as expected.
  * The item is tested and verified by onshore QA.
  * The item is deployed to Production.
