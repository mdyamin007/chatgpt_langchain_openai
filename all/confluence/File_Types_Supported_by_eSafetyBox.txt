title: File Types Supported by eSafetyBox
body: File Type (Extension)| Description  
---|---  
avi| Audio Video Interleaved Multimedia (Microsoft)  
bmp| Bitmap Image  
csv| Comma Separated Values  
doc| Word Document  
docx| Word Document 2007/2010  
flv| Flash Video (Adobe)  
gif| Graphic Interchange Format image  
htm| Hypertext Markup  
html| Hypertext Markup  
jpeg| Joint Photographic Experts Group Image  
jpg| Joing Photographic Experts Group Image  
mov| Movie/Video (Apple)  
mp3| MP3 audio  
mp4| MPEG-4 video  
odp| Open Document Presentation  
ods| Open Document Spreadsheet  
odt| Open Document Text  
pdf| Portable Document Format (Adobe)  
png| Portable Network Graphics Image  
ppt| PowerPoint Presentation  
pptx| PowerPoint Presentation 2007/2010  
rtf| Rich Text Format (Microsoft)  
swf| Showckwave Flash Video (Adobe)  
tif| Tagged Image File image  
tiff| Tagged Image File image  
txt| Plain text  
wdp| Windows Drawing Project  
wmv| Windows Media Video  
wpd| Word Perfect Document  
xls| Excel Spreadsheet  
xlsx| Excel Spreadsheet 2007/2010  
  
 _At least one file of each type needs to be tested. A collection of positive
and negative text cases (e.g. large files, .exe, .jsp files) are in the share:
dell-2100\Ben Chan\eSafetyBox\FileTypeSupportTestFiles  
For test automation, these files have to be added to the test/resources
directory of the Maven project in SVN._
