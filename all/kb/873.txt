Project Import:
Step 1: Open Papyrus Designer and go to File -> Import project --
Step 2: Following dialogue will be appear. Select Project location and press OK button
Step 3: Now scroll down new dialogue box and Deselect default.lbp and Ppde.prf file . Then finally press Rewrite selected button.
Step 4.1: Lets start to open new imported project. go to File -> Load Project
Step 4.2: Select project_name.prj file and finally open the project-
Project Export:
Step 1: go to File -> Export Project --
Step 2: Then Browse where you want to store the project
Step 3: Finally select OK button-
