3.Convert tiff or JPG to pseg
Open Overview Image Editor
Open the tiff or JPG file ( Go to File > open )
Go to image > properties.
Select inch option for size and 300 for resolution
Check Maintain aspect ratio and Maintain dimensions
Go to File > Save as
Save the pseg file in C:\ISIS\pseg directory
Select the TIFF in Compression and CMYK in Color Mode
